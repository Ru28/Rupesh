Hashedin by Delottie interview experience

I am a CSE undergradute currently in 4th year.I have applied in Hashedin by Delottie through on-campus placement drive for SDE-1 role.In this post I am going to share my interview experience for the same.

Online Assessments:

It was a 90 minute test on Codility.The test consisted of 3 coding question. level of defficulty of online assessments is easy-medium questions, I solved all questions,after I was getting shortlist for technical rounds.

each technical round is elimination round, hashedin conducted 2 technical round, 1 HR(fitment) round.

Round 1:

It was a 45 minute round. The interviewer asked me for a brief introduction followed by 2 programming questions which I had to code on a shared doc. one was about two sum problems which is leetcode easy level, second question is Group Anagrams which is leetcode medium.

  1. Two Sum
  2. Group Anagrams

After clearing first technical round, I am selected for second technical round. interview asked me two question first leetcode hard level question. Basically, question is reverse linkedlist in k-group. This question solved using recursive approach.first I explained my approach to interviewer.after I code it. second design database and all table should be in 3NF. Relate table using forign key, table should contain primary key, this regarding design database for University.

  1. Reverse linkedlist in k-group
  2. design University database

my second round was going well,I solved both questions and give best in interview, but I was unable to get call for HR round.

status: Rejected
